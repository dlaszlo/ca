Root CA Certificate

Root CA for self-hosted services. Install this certificate to trust HTTPS connections.

Download ca.pem

Installation

iPhone / iPad

  1. Tap Download above
  2. Settings → Profile Downloaded → Install
  3. Settings → General → About → Certificate Trust Settings → Enable
Step 3 is required! Without it, the certificate won't be trusted.

Android

  1. Tap Download above
  2. Settings → Security → Encryption & Credentials → Install a certificate → CA certificate

Chrome / Edge (Linux)

  1. Download ca.pem
  2. Run: certutil -d sql:$HOME/.pki/nssdb -A -t "C,," -n "Home Root CA" -i ca.pem

Firefox

  1. Settings → Privacy & Security → Certificates → View Certificates
  2. Authorities → Import → select ca.pem
  3. Check: Trust this CA to identify websites

macOS

  1. Download ca.pem
  2. Run: sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain ca.pem
